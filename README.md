# Лабораторна робота 4 
## Основні зміни
- **JSON-сховище замість FakeStore**: додано `JsonFileStudyStore`, який зберігає дані у `study_store.json` та створює файл з початковими (seed) даними при першому запуску.
- **Асинхронна взаємодія з даними**: Repository/Service/UI переведені на `async/await`, додано `IsBusy` для керування станом UI під час завантаження.
- **CRUD (частково/основне)**:
  - Додавання та видалення **предметів** (каскадно видаляються всі заняття предмета).
  - Додавання та видалення **занять** у межах вибраного предмета.
- **Режим редагування предмета (сутність 1-го рівня)**: на сторінці деталей предмета можна перейти в режим редагування полів (Edit/Save/Cancel) і зберегти зміни у JSON.
- **Пошук/фільтрація/сортування**: реалізовано для списку предметів і для списку занять на сторінці предмета.
- **Оновлений стиль UI**: додано глобальні стилі для кнопок, текстових полів, списків.
